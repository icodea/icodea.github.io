# Linux命令大全教程

​ Linux是一套免费使用和自由传播的类Unix操作系统，是一个基于POSIX和UNIX的多用户、多任务、支持多线程和多CPU的操作系统。它能运行主要的UNIX工具软件、应用程序和网络协议。它支持32位和64位硬件。Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。

​ Linux是一款免费的操作系统，用户可以通过网络或其他途径免费获得，并可以任意修改其源代码。这是其他的操作系统所做不到的。正是由于这一点，来自全世界的无数程序员参与了Linux的修改、编写工作，程序员可以根据自己的兴趣和灵感对其进行改变，这让Linux吸收了无数程序员的精华，不断壮大。

# 适用者

​ 本教程是专为Linux初学者而准备的，通过简单的步骤帮助他们来了解和学习Linux基础知识。 完成本教程的学习后，您将发现自己在Linux方面具有一定的专业知识水平，进而能更容易地学习Linux中更深层次的技术和知识。

# 前提条件

​ Linux是一个操作系统，所以如果你知道或熟悉其它类Unix系统或Window的批处理命令，那么将很容易学习Linux。如果没有了解这些，也没有关系，可以按我们提供的教程，从头到尾学习和执行就好。

# 一些Linux命令示例

  • ps命令 - Linux中的ps命令是Process Status的缩写。ps命令用来列出系统中当前运行的那些进程。ps命令列出的是当前那些进程的快照，就是执行ps命令的那个时刻的那些进程。
  • at命令 - 定时任务，指定一个时间执行一个任务，只能执行一次。
  • crontab命令（Linux定时任务） - 循环运行的例行性计划任务，linux系统则是由cron(crond) 这个系统服务来控制的。Linux 系统上面原本就有非常多的计划性工作，因此这个系统服务是默认启动的。
