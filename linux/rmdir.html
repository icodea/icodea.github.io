# rmdir

еңЁLinuxдёӯзҡ„rmdirжҳҜеёёз”Ёзҡ„е‘Ҫд»ӨпјҢиҜҘе‘Ҫд»Өзҡ„еҠҹиғҪжҳҜеҲ йҷӨз©әзӣ®еҪ•пјҢдёҖдёӘзӣ®еҪ•иў«еҲ йҷӨд№ӢеүҚеҝ…йЎ»жҳҜз©әзҡ„гҖӮ(жіЁж„ҸпјҢrm -r dirе‘Ҫд»ӨеҸҜд»ЈжӣҝrmdirпјҢдҪҶжҳҜжңүеҫҲеӨ§еҚұйҷ©жҖ§гҖӮ)еҲ йҷӨжҹҗзӣ®еҪ•ж—¶д№ҹеҝ…йЎ»е…·жңүеҜ№зҲ¶зӣ®еҪ•зҡ„еҶҷжқғйҷҗгҖӮ

# 1пјҺе‘Ҫд»Өж јејҸ

rmdir [йҖүйЎ№]вҖҰ зӣ®еҪ•вҖҰ

# 2пјҺе‘Ҫд»ӨеҠҹиғҪ

иҜҘе‘Ҫд»Өд»ҺдёҖдёӘзӣ®еҪ•дёӯеҲ йҷӨдёҖдёӘжҲ–еӨҡдёӘеӯҗзӣ®еҪ•йЎ№пјҢеҲ йҷӨжҹҗзӣ®еҪ•ж—¶д№ҹеҝ…йЎ»е…·жңүеҜ№зҲ¶зӣ®еҪ•зҡ„еҶҷжқғйҷҗгҖӮ

# 3пјҺе‘Ҫд»ӨеҸӮж•°

  • - pйҖ’еҪ’еҲ йҷӨзӣ®еҪ•dirnameпјҢеҪ“еӯҗзӣ®еҪ•еҲ йҷӨеҗҺе…¶зҲ¶зӣ®еҪ•дёәз©әж—¶пјҢд№ҹдёҖеҗҢиў«еҲ йҷӨгҖӮеҰӮжһңж•ҙдёӘи·Ҝеҫ„иў«еҲ йҷӨжҲ–иҖ…з”ұдәҺжҹҗз§ҚеҺҹеӣ дҝқз•ҷйғЁеҲҶи·Ҝеҫ„пјҢеҲҷзі»з»ҹеңЁж ҮеҮҶиҫ“еҮәдёҠжҳҫзӨәзӣёеә”зҡ„дҝЎжҒҜгҖӮ
  • -v, --verbose жҳҫзӨәжҢҮд»Өжү§иЎҢиҝҮзЁӢ

# 4пјҺе‘Ҫд»Өе®һдҫӢ
